# Library Management System (SQL Server)

## Description
This project is a Library Management System database created using SQL Server. It manages books, members, and book issue records.

## Database Tables
- Books
- Members
- IssueRecords

## Features
- Add Books
- Add Members
- Issue Book
- Return Book
- SQL JOIN Queries

## Technologies Used
- SQL Server
- Relational Database
- SQL Queries
